# Pet Gora Bets - MVP

Kurzanleitung:

- Node.js (>=18) installieren
- Abhängigkeiten: npm install
- Server starten: npm run start

Das Projekt ist ein MVP: Express-Server, SQLite-Schema (db/schema.sql), statische Seiten im /public Ordner.

Wichtig: Echtgeld-Wetten sind rechtlich reguliert. Dieses MVP nutzt virtuelle Coins.
